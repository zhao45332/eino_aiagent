# 企业级 FAQ 语料（以下 # 标题行不进入向量索引）

# 一、业务咨询类

1. 贵公司主要提供哪些企业级服务？答：我司聚焦企业全生命周期服务，核心业务涵盖：企业数字化转型解决方案（含系统搭建、数据迁移、定制开发）、企业运维服务（服务器运维、网络安全、数据备份）、企业管理咨询（组织架构优化、流程梳理、合规辅导）、云服务部署（公有云、私有云、混合云搭建与运维），可根据企业规模（小型、中型、大型）及行业特性，提供定制化服务套餐。

2. 服务的收费标准是什么？是否有优惠政策？答：我司收费实行「基础套餐+定制增值」模式，具体费用根据服务类型、服务周期、企业规模及定制需求核算，无统一固定报价。优惠政策：长期合作客户（服务周期≥1年）可享受8.5-9折优惠；新客户首次合作可赠送1次免费服务巡检；集团客户、多业务联动合作可申请专属定制优惠，详情可联系商务顾问咨询。

3. 不同行业的企业，服务方案有差异吗？答：有显著差异。我司针对制造业、金融业、零售业、互联网、事业单位等不同行业，结合其行业合规要求、业务场景特点（如制造业的生产流程数字化、金融业的数据安全合规），定制专属服务方案，避免「一刀切」，确保服务贴合企业实际需求，提升落地效果。

4. 服务的交付周期大概是多久？答：交付周期根据服务类型而定：标准化服务（如基础运维、简单咨询）1-7个工作日；常规定制服务（如小型系统开发、流程梳理）1-3个月；大型复杂项目（如全流程数字化转型、大型云部署）3-12个月。具体交付周期会在合作合同中明确；若企业有紧急需求，可协商加急处理，加急费用另行核算。

# 二、账号与权限类

1. 企业账号如何注册与开通？答：企业账号注册需由企业授权负责人（法人或指定对接人）发起。流程：登录我司官网点击「企业注册」，填写企业全称、统一社会信用代码、注册地址等基础信息；上传企业营业执照、法人身份证正反面照片（加盖企业公章）；填写授权对接人信息，绑定手机号并完成短信验证；我司审核人员将在1-2个工作日内完成审核，审核通过后账号自动开通，相关登录信息将发送至对接人预留手机号。

2. 企业账号可设置多个子账号吗？权限如何分配？答：可以设置多个子账号，支持分级权限管理。企业主账号（授权负责人）可登录后台新建子账号并分配权限：管理员权限可管理所有子账号、查看全部服务数据、审批服务申请、修改企业信息；对接人权限可查看对应服务进度、提交服务需求、接收通知提醒；只读权限仅可查看服务相关数据，无法操作修改。子账号数量无上限，权限可随时调整。

3. 账号密码忘记了，如何重置？答：子账号：联系企业主账号管理员，由管理员在后台「子账号管理」中重置密码，重置后可自行修改。主账号：点击官网登录页面「忘记密码」，输入企业全称、统一社会信用代码及预留手机号，完成短信验证后可重置密码；若无法验证，可联系客服提供企业相关资质，核实后协助重置。

4. 企业信息变更（如法人、地址），如何更新账号信息？答：须由主账号管理员操作：登录企业账号后台进入「企业信息管理」；点击「信息修改」，上传变更后的证明材料（如法人变更需上传新法人身份证、变更通知书；地址变更需上传新营业执照）；提交修改申请，我司审核人员将在1个工作日内完成审核，审核通过后信息自动更新；特殊情况可联系客服协助处理。

# 三、服务支持类

1. 服务过程中，如何反馈问题或提出需求？答：提供4种渠道：企业账号后台「服务工单」提交问题与截图、选择紧急程度，工作人员1-2小时内响应；专属对接人（商务顾问及技术对接人）电话、微信；客服热线拨打全国统一热线（XXX-XXXXXXX），工作日9:00-18:00专人接听；企业邮箱发送至 service@xxx.com，24小时内回复处理结果。

2. 服务出现故障或异常，如何处理？答：紧急故障（影响正常运营）：拨打专属技术对接人或客服紧急热线，安排技术人员1小时内介入，重大故障4小时内解决。一般故障：通过工单、邮箱反馈，2小时内响应，24小时内解决。处理完成后出具故障分析报告及预防方案。

3. 服务结束后，是否提供后续保障？答：标准化服务结束后提供1个月免费售后（咨询、小型异常）；定制化项目结束后提供3-6个月免费售后（技术支持、系统调试、数据维护）；长期合作客户可享受全年免费巡检（每季度1次）、技术培训、优先响应等；升级与新增需求可享受老客户专属优惠。

4. 如何查询服务进度及历史记录？答：登录企业账号后台进入「服务管理」，可查看所有服务进度（审核中、实施中、已完成）、负责人、预计完成时间；点击对应服务可查看历史操作记录、反馈记录、处理结果及相关文档，记录长期留存便于追溯。

# 四、合作与合同类

1. 企业合作的流程是什么？答：五步：需求沟通（商务顾问了解需求、初步方案）；方案定制（技术团队制定详细方案并与企业确认）；合同签订（支付预付款，一般30%-50%）；服务实施（专项团队推进、定期同步进度）；验收交付（企业验收、支付尾款、交付文档成果与售后保障）。

2. 合同签订方式有哪些？是否支持电子合同？答：支持纸质合同（邮寄签字盖章后寄回，双方各留存1份）与电子合同（通过国家认可的电子合同平台签订，与纸质合同同等法律效力，可在账号后台查看下载）。电子合同由授权负责人完成身份验证后在线签字即可。

3. 预付款和尾款的支付方式有哪些？答：支持银行对公转账（提供正式对公账户）、企业账号后台对公扫码/企业网银、大型项目可支持汇票支票等（需提前与商务确认）。支付完成后3个工作日内开具等额增值税发票（专票/普票按企业需求）。

4. 合同可以变更或解除吗？流程是什么？答：可以，须双方协商一致。变更：提出变更需求，确认后签订补充协议，与原合同同等效力。解除：不可抗力、协商一致或违约等情形可申请解除，签订解除协议明确款项结算与成果交付；单方违约解除按合同承担违约责任。

# 五、合规与安全类

1. 贵公司的服务是否符合企业合规要求？答：我司服务遵循《网络安全法》《数据安全法》《个人信息保护法》等；金融、医疗、政务等行业额外符合行业监管要求。合作前提供合规方案说明；服务过程中定期开展合规检查，可配合企业合规审计。

2. 企业数据安全如何保障？答：技术：加密存储、访问控制、入侵检测等；管理：数据安全管理制度、全流程规范、人员保密协议；合规：不收集泄露无关数据，服务结束可按企业要求删除或返还；应急：数据安全应急预案，事件第一时间介入。

3. 服务过程中，会涉及企业核心机密泄露吗？答：不会故意为之；合作前签订保密协议（范围含核心数据、业务流程、技术方案等，保密期限一般至企业书面同意前）；服务中仅授权人员在服务范围内使用，严禁私自留存传播；结束后回收文档数据、删除备份。若因我司原因泄露，按协议与法律承担责任。
